## sms-bomber
### sms bomber with Termux.



#### 1- install updates

* `apt update  -y`
* `apt upgrade -y`


#### 2- Install prerequisites

* `pkg install git`
* `pkg install python`



#### 3- Install tools

* `git clone https://github.com/NiREvil/sms-bomber`



### 4- Tool directory

* `cd sms-bomber`



### 5- run

* `python sms-Bomber.py`
